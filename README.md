### 计算器

> 推荐无基础的大一新生和实力稍弱的大二同学选择此题

  <br/>

*Feature*

支持表达式计算的计算器

表达式可包含 括号 加减乘除

  <br/>

*pluses*

多种临界情况判断 请避免程序在极端情况下出现崩溃或者难以使用的情况

可在表达式中嵌套入 sin cos

请让你的表达式尽可能的美观易懂

  <br/>

*key words*

逆波兰表达式

  <br/>
